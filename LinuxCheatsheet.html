Linux Cheatsheet

Basic Commands


ls                   # List files in the directory
ls -a                # List all files including hidden files
ls -l                # List files in long format
cd /path/to/dir      # Change directory
pwd                  # Print working directory
mkdir dir            # Create a new directory
rmdir dir            # Remove an empty directory
rm file              # Remove a file
rm -r dir            # Remove a directory and its contents recursively
cp file1 file2       # Copy file1 to file2
cp -r dir1 dir2      # Copy dir1 to dir2 recursively
mv file1 file2       # Move/rename file1 to file2
touch file           # Create an empty file or update the timestamp

File Viewing and Editing


cat file             # View file contents
less file            # View file contents one page at a time
head file            # View the first 10 lines of a file
head -n 20 file      # View the first 20 lines of a file
tail file            # View the last 10 lines of a file
tail -n 20 file      # View the last 20 lines of a file
nano file            # Edit a file using nano editor
vi file              # Edit a file using vi editor

File Permissions


chmod 755 file       # Change file permissions to rwxr-xr-x
chmod u+x file       # Add execute permission for the user
chown user:group file # Change file owner and group

Process Management


ps                   # Display current processes
ps aux               # Display all running processes
top                  # Display real-time process information
kill PID             # Kill process with given PID
kill -9 PID          # Force kill process with given PID

Networking


ifconfig             # Display network configuration (older systems)
ip addr show         # Display IP addresses (modern systems)
ping hostname        # Ping a host to check connectivity
netstat -tuln        # Display listening ports
curl http://url      # Fetch a URL's content
wget http://url      # Download a file from a URL

Disk Usage


df -h                # Display disk space usage in human-readable format
du -sh dir           # Display disk usage of a directory

Package Management


# Debian/Ubuntu
sudo apt update                    # Update package lists
sudo apt upgrade                   # Upgrade all packages
sudo apt install package           # Install a package
sudo apt remove package            # Remove a package
sudo apt autoremove                # Remove unused packages

# Red Hat/CentOS
sudo yum update                    # Update all packages
sudo yum install package           # Install a package
sudo yum remove package            # Remove a package
sudo yum clean all                 # Clean up cached packages

User Management


sudo adduser username                # Add a new user
sudo passwd username                 # Change user password
sudo deluser username                # Delete a user
sudo usermod -aG group username      # Add user to a group

System Information


uname -a                 # Display system information
uname -r                 # Display kernel version
lsb_release -a           # Display distribution-specific information
free -h                  # Display memory usage in human-readable format
df -h                    # Display disk usage in human-readable format

Searching and Finding Files


find /path -name filename             # Find files by name
grep "pattern" file                   # Search for a pattern in a file
grep -r "pattern" /path               # Search for a pattern recursively in directory

File Compression and Archiving


tar -czvf archive.tar.gz dir          # Create a gzip compressed tar file
tar -xzvf archive.tar.gz              # Extract a gzip compressed tar file
zip archive.zip file1 file2           # Create a zip file
unzip archive.zip                     # Extract a zip file

System Monitoring


htop                                  # Interactive process viewer (alternative to top)
iostat                                # CPU and I/O statistics
vmstat                                # System performance and virtual memory statistics

Permissions and Ownership


chmod 644 file                        # Change permissions to rw-r--r--
chown user file                       # Change owner of file
chgrp group file                      # Change group of file

Crontab (Scheduling Tasks)


crontab -e                            # Edit crontab for current user
crontab -l                            # List crontab entries for current user

SSH (Secure Shell)


ssh user@host                         # Connect to host as user
ssh -i /path/to/key user@host         # Connect using a specific SSH key
scp file user@host:/path/to/dest      # Copy file to remote host using SCP
